Week 1 Entry

How did you approach mobile-first design?
I started by designing for small screens using flexible layouts and percentages, then added media queries for larger devices.

Most useful concept?
Learning how Flexbox simplifies layout control and responsive design.

Most challenging part?
Understanding the CSS box model and how padding and margins affect total width.
